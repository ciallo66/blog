# ES6语法

### Array.from( [空位一] , [空位二])
1. **空位一**:【必填：伪数组或长相怪异的对象】
2. **空位二**:【选填：挨个处理的加工函数】
    >(currentValue,index)=>{}

```ts
//创建一个新的数组
const starObjects = Array.from({length:100}, (_, i) => ({        //length:100是让其预留100个空位，因为没有数据用_占位
    id: i,
    left: `${Math.random() * 100}%`,
    top: `${Math.random() * 100}%`
}));
```
