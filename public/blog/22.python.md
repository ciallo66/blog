# python的学习
>Python 注重开发效率，优势在于语法简洁、生态丰富，可以通过大量成熟库快速完成复杂功能开发，减少重复代码，提高开发效率。
>目前 Python 主要应用于人工智能、数据分析、自动化脚本、爬虫、Web 后端开发等领域，尤其在 AI 等智能体研发有独有的优势。

## 一.基础语法
### 1. 变量与数据类型
Python 不需要提前声明变量类型，直接赋值即可。常用数据类型主要有：
| 类型    | 作用   |
| ----- | ---- |
| int   | 整数   |
| float | 小数   |
| str   | 字符串  |
| bool  | 判断真假 |
| None  | 表示空值 |

**注意：**
* Python 是动态类型语言，变量类型会根据赋值自动变化。
* 不同类型之间经常需要转换，例如字符串转数字。

### 2. 条件判断与循环
Python 通过条件和循环控制程序执行流程。
| 功能               | 作用         |
| ---------------- | ---------- |
| if / elif / else | 根据条件执行不同代码 |
| for              | 遍历数据       |
| while            | 满足条件持续执行   |
| break            | 结束循环       |
| continue         | 跳过当前循环     |

**注意：**
* Python 使用缩进表示代码层级，不使用大括号。
* 循环主要用于处理列表、文件、数据等批量操作。

### 3.输入输出
- input 
表示输入
- print
表示输出


## 二.数据结构
Python 常用数据结构主要包括列表（list）、字典（dict）、元组（tuple）、集合（set），用于不同场景的数据存储和处理。(只有列表和元组可以进行下标访问)

### 1. 列表（list）
列表用于存储一组有序、可修改的数据，是开发中最常用的数据结构。

基本格式：
```python
list_name = [数据1, 数据2, 数据3]
```

常用操作：
```python
list.append()    # 添加数据
list.remove()    # 删除数据
list[index]      # 根据索引获取数据
list[index] = x  # 修改数据
```
常用于保存用户列表、商品列表等需要遍历和修改的数据。

### 2. 字典（dict）
字典通过 key-value（键值对）保存数据，适合根据标识快速获取信息。

基本格式：
```python
dict_name = {
    "key": "value"
}
```

常用操作：
```python
dict["key"]          # 获取数据
dict["key"] = value  # 修改或添加数据
dict.get("key")      # 安全获取数据
```
实际开发中大量使用，例如接口数据、用户信息、配置文件等。
### 3. 元组（tuple）
元组用于保存不可修改的数据。

基本格式：
```python
tuple_name = (数据1, 数据2)
```

特点：
* 有序
* 不可修改

适合保存固定数据。
### 4. 集合（set）
集合用于存储不重复的数据，常用于去重。

基本格式：
```python
set_name = {数据1, 数据2}
```

常用操作：
```python
set.add()       # 添加数据
set.remove()    # 删除数据
```
### 5. 实际开发选择
| 数据结构  | 主要用途          |
| ----- | ------------- |
| list  | 管理一组可变化的数据    |
| dict  | 根据 key 快速查询数据 |
| tuple | 保存固定不变的数据     |
| set   | 数据去重          |

实际开发中最常用的是 **list 和 dict**，尤其是在接口开发、数据处理、业务逻辑中大量出现。
