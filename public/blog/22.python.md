# python的学习
>Python 注重开发效率，优势在于语法简洁、生态丰富，可以通过大量成熟库快速完成复杂功能开发，减少重复代码，提高开发效率。  
>目前 Python 主要应用于人工智能、数据分析、自动化脚本、爬虫、Web 后端开发等领域，尤其在 AI 等智能体研发有独有的优势。

## 一.基础语法
### 1. 变量与数据类型
Python 不需要提前声明变量类型，直接赋值即可。常用数据类型主要有：
| 类型    | 作用   |
| ----- | ---- |
| int   | 整数   |
| float | 小数   |
| str   | 字符串  |
| bool  | 判断真假 |
| None  | 表示空值 |

**注意：**
* Python 是动态类型语言，变量类型会根据赋值自动变化。
* 不同类型之间经常需要转换，例如字符串转数字。

### 2. 条件判断与循环
Python 通过条件和循环控制程序执行流程。
| 功能               | 作用         |
| ---------------- | ---------- |
| if / elif / else | 根据条件执行不同代码 |
| for              | 遍历数据       |
| while            | 满足条件持续执行   |
| break            | 结束循环       |
| continue         | 跳过当前循环     |

**注意：**
* Python 使用缩进表示代码层级，不使用大括号。
* 循环主要用于处理列表、文件、数据等批量操作。

### 3.输入输出
- input 
表示输入
- print
表示输出


## 二.数据结构
Python 常用数据结构主要包括列表（list）、字典（dict）、元组（tuple）、集合（set），用于不同场景的数据存储和处理。(只有列表和元组可以进行下标访问)

### 1. 列表（list）
列表用于存储一组有序、可修改的数据，是开发中最常用的数据结构。

基本格式：
```python
list_name = [数据1, 数据2, 数据3]
```

常用操作：
```python
list.append()    # 添加数据
list.remove()    # 删除数据
list[index]      # 根据索引获取数据
list[index] = x  # 修改数据
```
常用于保存用户列表、商品列表等需要遍历和修改的数据。

### 2. 字典（dict）
字典通过 key-value（键值对）保存数据，适合根据标识快速获取信息。

基本格式：
```python
dict_name = {
    "key": "value"
}
```

常用操作：
```python
dict["key"]          # 获取数据
dict["key"] = value  # 修改或添加数据
dict.get("key")      # 安全获取数据
```
实际开发中大量使用，例如接口数据、用户信息、配置文件等。
### 3. 元组（tuple）
元组用于保存不可修改的数据。

基本格式：
```python
tuple_name = (数据1, 数据2)
```

特点：
* 有序
* 不可修改

适合保存固定数据。
### 4. 集合（set）
集合用于存储不重复的数据，常用于去重。

基本格式：
```python
set_name = {数据1, 数据2}
```

常用操作：
```python
set.add()       # 添加数据
set.remove()    # 删除数据
```
### 5. 实际开发选择
| 数据结构  | 主要用途          |
| ----- | ------------- |
| list  | 管理一组可变化的数据    |
| dict  | 根据 key 快速查询数据 |
| tuple | 保存固定不变的数据     |
| set   | 数据去重          |

实际开发中最常用的是 **list 和 dict**，尤其是在接口开发、数据处理、业务逻辑中大量出现。

## 三.函数与模块
函数用于封装一段可重复执行的代码，提高代码复用性；模块用于组织和复用代码，将不同功能拆分到不同文件中。

## 1. 函数定义与参数
函数通过 `def` 定义，使用参数接收外部传入的数据，通过返回值输出结果。
基本格式：
```python
def 函数名(参数):
    执行代码
    return 返回值
```

参数常见类型：
| 类型       | 说明           |
| -------- | ------------ |
| 位置参数     | 按顺序传递参数      |
| 默认参数     | 参数没有传入时使用默认值 |
| *args    | 接收多个位置参数     |
| **kwargs | 接收多个键值参数     |

---

## 2. 变量作用域

变量的有效范围由定义位置决定：

* 局部变量：函数内部创建，只能在函数内部使用。
* 全局变量：函数外部创建，可以被多个函数访问。

实际开发中尽量减少全局变量，避免数据被随意修改。

---

## 3. 模块与库

模块就是一个 Python 文件，可以将代码拆分管理，通过 `import` 引入其他模块使用。

常见方式：

* 导入内置模块：

  * Python 自带的功能模块，例如时间、文件处理等。
* 自定义模块：

  * 将自己编写的代码文件引入其他项目。
* 第三方库：

  * 通过包管理工具安装后使用，例如网络请求、数据处理等。

Python 的模块机制让代码更容易维护和复用，是实际项目开发中的基础能力。

## 四.面向对象编程（OOP）
Python 是一种支持面向对象编程的语言，OOP 主要用于将数据和功能封装在一起，让代码结构更加清晰，提高代码复用性和维护性。在实际项目中，常用于管理用户、商品、订单等具有属性和行为的业务对象。
## 1. 类与对象
类是创建对象的模板，对象是类创建出来的具体实例。
基本格式：
```python
class 类名:
    def 方法名(self):
        代码
```

主要概念：
| 内容         | 说明                  |
| ---------- | ------------------- |
| 类（class）   | 定义对象的结构和功能          |
| 对象（object） | 根据类创建的具体实例          |
| 属性         | 对象保存的数据，例如用户名称、商品价格 |
| 方法         | 对象可以执行的操作           |

## 2. 构造函数与封装
构造函数 `__init__()` 会在创建对象时自动执行，主要用于初始化对象属性。
封装是将数据和相关操作放在类内部，通过方法控制数据访问，避免外部直接修改对象内部状态，提高代码安全性和可维护性。

## 3. 继承与多态
面向对象中的重要特性：
| 特性 | 作用                     |
| -- | ---------------------- |
| 继承 | 子类继承父类属性和方法，减少重复代码     |
| 多态 | 不同对象调用同一个方法时，可以表现不同的行为 |

继承适合处理具有相同特征的对象，例如管理员和普通用户都属于用户，可以复用公共功能。

## 五.文件与异常处理
文件操作和异常处理是 Python 实际开发中经常使用的内容，主要用于数据持久化、读取配置以及提高程序运行稳定性。

## 1. 文件操作
Python 可以通过文件操作实现数据的读取和保存，常见操作包括：
| 操作   | 作用       |
| ---- | -------- |
| 打开文件 | 获取文件对象   |
| 读取文件 | 获取文件中的数据 |
| 写入文件 | 保存数据到文件  |

基本格式：
```python
with open("test.txt") as f:
    data = f.read()
```

常用方式：
* `read()`：读取文件内容
* `write()`：写入数据
* `with`：自动关闭文件，避免资源占用

实际开发中常用于读取配置文件、日志处理、数据保存等场景。
## 2. 异常处理
异常处理用于捕获程序运行中的错误，避免程序因为异常直接终止。
基本格式：
```python
try:
    执行代码
except:
    错误处理
```

主要结构：
| 关键字     | 作用         |
| ------- | ---------- |
| try     | 执行可能出错的代码  |
| except  | 捕获并处理异常    |
| finally | 无论是否异常都会执行 |

实际开发中常用于处理文件不存在、网络请求失败、数据格式错误等问题，提高程序稳定性。
