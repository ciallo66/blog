# 一些语法糖

### 1.v-model(数据双向绑定)
```js
//原来
:value="name"
@input="name = $event.target.value"

//现在
v-model="name"
```

### 2.赋值解构
```js
//原来
const operator = { name: '阿米娅', level: 90 }
const name = operator.name; const level = operator.level

//现在
const { name, level, age = 14 } = operator;        //age默认值
```

### 3.可选链(?.)
```js
if (user && user.info && user.info.avatar)

user?.info?.avatar
```

### 4.三元符(a?b:c)    
```js
//原来
if(){}else{}
//现在
条件 ? 满足就执行我 : 不满足就执行我
```

### 5.对象简写
- **变量名和key相同时**
```js
//原来
const operator = {name: name};
//现在
const operator = {name};
```

### 6.扩展运算符(...)
- 情况一
```js
//原来
arr1.concat(arr2)
//现在
const allOperators = [...oldList, ...newList]    //两个数组直接合并
```
- 情况二
```js
//防止连坐，b想直接复制a的数据，但是改b的数据，a也跟着变，使用 ...a 传递即可
const a = { name: '阿米娅', level: 1 };
const b = { ...a };

```

### 7.模版字符串(` `)
```js
//原来
const str1 = '获得 ' + rarity + ' 星干员 ' + name;
//现在
`获得 ${rarity} 星干员 ${name}`
```

### 8.箭头函数
```js
//以前
const add = function (x, y) {
    return x + y;
};
//现在
const add = (x, y) => {
    return x + y;
};
//如果是一行代码
const add = (x, y) => x + y;
```
