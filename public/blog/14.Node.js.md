## 2.fs模块
```js
//导入 fs 模块
const fs = require('fs')
```
### (1)写入文件
**语法**：` fs.writeFile(file, data, [options], callback)`  
**参数说明**:
- file 文件名
- data 待写入数据
- options 选项配置 (`{flag:'a'}`,w:写入,a:追加,r:读取)
- callback 回调 

**注意**:writeFile是异步，需要写同步用writeFileSync()

```js
fs.writeFile('./座右铭.txt','\r\n海纳百川，有容乃大。',err => {     // \r\n是换行
    if(err){
        console.log('写入失败');
        return;
    }
    console.log('写入成功');
});

```
