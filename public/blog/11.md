### 3.dompurify库(防xss)
```ts
//安装
npm install dompurify

//导入
import DOMPurify from 'dompurify'

//通过其他手段获取到的html，getHtml
newHtml =  DOMPurify.sanitize(getHtml)
//newHtml就是经过过滤的html。不仅剔除 <script>等恶意脚本标签，还会彻底剥离标签中残留的恶意事件属性（如onclick）以及危险的伪协议链接，
```
