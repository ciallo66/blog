# Fastapi

## 一.初始化环境
```bash
# 1. 创建项目
mkdir FastAPI_Project

# 2. 创建虚拟环境（仅第一次）
python -m venv .venv

# 3. 激活虚拟环境
.venv\Scripts\activate

# 4. 安装 FastAPI
pip install fastapi

# 5. 安装 Uvicorn
pip install uvicorn

# 6. 保存项目依赖（推荐）
pip freeze > requirements.txt
```

## 二.基础语法
### 1.常用路由类装饰器
| 装饰器              | HTTP 方法 | 作用           | 常见用途           |
| ---------------- | ------- | ------------ | -------------- |
| `@app.get()`     | GET     | 查询数据         | 获取用户、获取文章列表    |
| `@app.post()`    | POST    | 新增数据         | 登录、注册、新建数据     |
| `@app.put()`     | PUT     | 修改整个资源       | 修改用户全部信息       |
| `@app.patch()`   | PATCH   | 修改部分资源       | 修改昵称、头像等部分字段   |
| `@app.delete()`  | DELETE  | 删除数据         | 删除用户、删除文章      |
