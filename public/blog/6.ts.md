## 二.ts专属类型
### 1.any
**作用**：表示任何类型，使用时不需要类型检查
**注意**：可赋值给任意类型的变量，可能会影响其他变量，少用

### 2.unknow
**作用**：安全的表示任何类型，使用前需要类型检查
#### (1)进行if判断
```ts
if(typeof a === 'string'){
    x = a
}
```
#### (2)断言
```ts
//第一种写法
x = a as string
//第二种写法
x = <string> a
```





### 3. never
**作用**：不能有值
1. 不同于限制变量
2. 一般由ts推断出来的
3. 限制函数返回值，该函数不能正常结束

### 4. void
**作用**：限制函数返回值，可以是显式和隐式undefined，结果不可被调用
1. 从语法上讲：函数是可以返回 undefined 的，至于显示返回，还是隐式返回，这无所谓！
2. 从语义上讲：函数调用者不应关心函数返回的值，也不应依赖返回值进行任何操作！即使返回了 undefined 值。

### 接口
- 在src目录下中的types/index.ts里
``` ts
 export interface PersonInter{            //接口的定义，目标内容和接口不一致时会报错
    name:string,
    id:string,
    age:number
}

export type Persons = PersonInter[]         //自定义接口，可以直接调用Persons
```
